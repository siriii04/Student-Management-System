# Student Management System (Java)

A simple **Java-based Student Management System** that allows users to manage student records such as adding, viewing, searching, and deleting students.  
This project is designed for **learning core Java concepts**, **object-oriented programming**, and basic project structure.

---

## 🚀 Features

- Add new student details  
- View all students  
- Search student by ID  
- Delete student records  
- Optional GUI support using Java Swing  
- Stores student data locally in text files  

---

## 🛠 Technologies Used

- Java (JDK 17)  
- Java Collections (`ArrayList`)  
- Java Swing (GUI)  
- Git & GitHub
